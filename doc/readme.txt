To run OpenAkka:
Run Task3 JAR file and include the OpenAkka class and all the Akka JAR libraries in the classpath argument. Also specify the port number as an argument.
You can also run OpenAkka from within an IDE.

To run the Cakes program.
Specify how many instances of Bob you want on line 164 of Cakes.java
Map each Bob to an IP address and port number on line 172 of Cakes.java
Start the main method inside the Cakes class.
